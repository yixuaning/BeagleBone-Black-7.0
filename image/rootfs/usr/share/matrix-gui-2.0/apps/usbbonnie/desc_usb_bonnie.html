USB Mass Storage Class Bonnie++ Benchmark Overview

Purpose:

The USB MSC Bonnie++ benchmark is used to measure filesystem and block I/O performance on a USB Mass Storage Device. The application will detect an inserted USB Mass Storage Device, run Bonnie++ on the first mounted filesystem partition, and display the results.

Warning: Bonnie++ is optimized to work with ext2 and ext3 file systems and should run within a couple of minutes. If you use a VFAT (Windows) filesystem, it can take over 30 minutes to execute.

Additional Information

Output: Bonnie++ throughput results

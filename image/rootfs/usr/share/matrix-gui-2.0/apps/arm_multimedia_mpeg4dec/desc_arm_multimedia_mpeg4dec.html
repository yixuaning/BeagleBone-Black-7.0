ARM MPEG-4 Video Decode

This example demonstrates video decoding capability of Cortex-A8 core using MPEG-4 codec. The Multimedia framework used is GStreamer and gst-ffmpeg plug-in is used to call into open source FFmpeg codec libraries. The data file resolution has been modified to retain the original aspect ratio of 16:9.

Multimedia Pipeline

gst-launch
 |_ mpeg4videoparse
   |_ ffdec_mpeg4
     |_ v4l2/fbdev sink

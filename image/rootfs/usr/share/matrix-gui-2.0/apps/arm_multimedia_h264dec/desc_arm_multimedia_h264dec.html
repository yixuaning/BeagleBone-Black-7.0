ARM H.264 Video Decode

This example demonstrates video content decoding capability of Cortex-A8 core using H.264 codec. The Multimedia framework used is GStreamer and gst-ffmpeg plug-in is used to call into open source FFmpeg codec libraries.

Multimedia Pipeline

gst-launch
 |_ h264parse
   |_ ffdec_h264
     |_ v4l2/fbdev sink
